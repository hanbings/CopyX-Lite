这是一个“4小时挑战”作品

“4小时挑战”指的是4个小时之内完成开发的实用软件

CopyX-Lite是一个简单的剪切板记录器🗄🗄🗄

它能显示你过去一段时间的复制记录🍔🍔🍔

![](https://s2.ax1x.com/2020/01/25/1ZqU3Q.png)
