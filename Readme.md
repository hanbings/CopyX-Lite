这是一个“4小时挑战”作品

“4小时挑战”指的是4个小时之内完成开发的实用软件

CopyX-Lite是一个简单的剪切板记录器🗄🗄🗄

它能显示你过去一段时间的复制记录🍔🍔🍔

![](https://s2.ax1x.com/2020/01/25/1ZqU3Q.png)

使用的包有：

MaterialDesignColors.1.2.0

MaterialDesignThemes.2.6.0

已知BUG：

☢💀在获取剪切板的同时，如果按下清理，它将会停止工作

☢💀只能获取文本，当剪切板存在图片的同时，可能它会停止工作

☢💀剪切板内允许存在同样的文本
